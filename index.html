Class 1 Summary

In my first HTML class, I learned about the basic structure of an HTML document and how to work with Visual Studio for coding. Here's a summary of what I covered:

1. HTML Structure:

Every HTML document has a basic structure that includes key tags like <html>, <head>, and <body>. These elements define how the document is organized.

2. Title:

The title tag is placed inside the <head> section of an HTML document and defines the title of the webpage, which appears on the browser tab.

3. Head Tag:

The head tag contains metadata like the title, links to stylesheets, and scripts.

4. Body:

The "body" tag contains all the visible content of the webpage, such as text, images, and links.

5. Using Visual Studio:

I learned how to use Visual Studio as an editor for writing HTML code. I now know how to open a folder, create an HTML file, and start writing code within it

6. Adding Images:

I also learned how to add images to a webpage using the "img" tag. The tag requires the src attribute to specify the image's location and the alt attribute to describe the image in case it doesn't load.
